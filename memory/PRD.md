# Гильдия — Мобильное приложение
## PRD (Product Requirements Document)

### Описание
Мобильное приложение для клуба настольных ролевых игр «Гильдия». RPG-тематика с тёмной темой и золотыми акцентами, максимально приближенная к сайту guildkhv.com.

### Реализованные модули (v1.0)

#### Авторизация
- Регистрация: никнейм (латиница, 3+ символов), email, пароль (6+ символов)
- Вход: по никнейму или email + пароль
- JWT-токены (72 часа), хранение в AsyncStorage
- Автоматический вход при наличии токена

#### Каталог игр (Сессии)
- Список игр с карточками: название, описание, дата, длительность, мастер, локация, свободные места
- Детальный просмотр с модальным окном
- Запись на игру с подтверждением
- Отмена записи
- Бейдж занятости мест (зелёный/жёлтый/красный)
- Список участников

#### Мои записи
- Секция активных записей с деталями и кнопками «Мастеру» / «Отменить»
- Секция прошлых записей со статусом и оценкой
- Pull-to-refresh + автообновление при фокусе

#### Чат с мастером
- Открывается из активной записи через кнопку «Мастеру»
- Заголовок с именем мастера и названием игры
- Отправка текстовых сообщений
- **Отправка голосовых сообщений** (запись → реальная загрузка на сервер через веб-сессию)
- **Отправка изображений** (выбор из галереи → реальная загрузка)
- **Отправка документов** (выбор файла → загрузка)
- **Воспроизведение голосовых** (play/pause + прогресс-бар)
- **Просмотр изображений** (полноэкранный modal при нажатии)
- Бабблы сообщений (свои справа, мастера слева) с аватарами
- Временные метки, статус прочтения (✓✓)
- Автоопрос новых сообщений (polling каждые 5 секунд)
- Пустое состояние «Начните общение»
- Меню прикрепления: фото / документ

#### Кошелёк
- Карточка баланса с общей суммой
- Статус и кэшбэк
- Статистика: получено / потрачено / часы абонемента
- Фильтры: все / начисления / списания / награды
- История транзакций

#### Рейтинг (Лидерборд)
- Вкладки: Сессии / Золото / Достижения
- Ранжированный список с иконками для топ-3
- Подсветка текущего пользователя

#### Профиль
- RPG-карточка с аватаром, статусом
- Сетка статов: золото, сессии, записи, часы
- Прогресс-бар до следующего статуса
- Список достижений
- Мастера Гильдии (модальное окно с карточками мастеров)
- Кнопка выхода

### Технический стек
- **Backend**: FastAPI + MongoDB
- **Frontend**: Expo React Native (SDK 54)
- **Auth**: JWT (PyJWT + bcrypt)
- **Навигация**: expo-router (file-based) + bottom tabs
- **Анимации**: react-native-reanimated
- **Шрифты**: Cinzel (заголовки), EB Garamond (текст)
- **Цвета**: #0e0c15 (фон), #c9a96e (золото), #7c6bc4 (акцент)
